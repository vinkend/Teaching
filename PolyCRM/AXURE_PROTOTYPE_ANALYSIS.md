# 保利CRM会员管理系统 - 原型页面设计细节提取

## 1. 会员.html - 会员管理列表

### 页面布局
- **导航栏**：顶部导航，显示"Poly 保利会员"标题及用户信息下拉
- **左侧菜单栏**：多级折叠菜单，包含积分、优惠券、储值、营销、档案、设置、报表等模块
- **内容区域**：
  - 搜索筛选区（上方）
  - 操作按钮区（查询、重置、新增会员、导出）
  - 会员列表表格（下方）

### 表单字段（搜索筛选条件）
| 字段 | 类型 | 说明 |
|-----|------|------|
| 关键字 | 文本框 | 搜索会员信息 |
| 注册日期 | 日期范围 | 从-至日期选择 |
| 性别 | 下拉选择 | 单选男/女等 |
| 会员等级 | 下拉选择 | 选择等级 |
| 会员状态 | 下拉选择 | 选择状态 |
| 注册渠道 | 下拉选择 | 小程序、APP等 |
| 注册酒店 | 下拉选择 | 选择酒店 |

### 表格列信息
| 列号 | 列名 | 数据类型 | 说明 |
|------|------|--------|------|

| 2 | 姓名 | String | 会员姓名 |
| 3 | 注册手机号 | String | 电话号码 |
| 4 | 邮箱 | String | 可选 |
| 5 | 会员卡号 | String | 卡号，如PM00199001 |
| 6 | 会员等级 | String | 银卡、金卡等 |
| 7 | 永久等级 | String | 是/否 |
| 8 | 注册渠道 | String | 小程序、APP、web等 |
| 9 | 注册酒店 | String | 酒店名称 |
| 10 | 性别 | String | 男/女 |
| 11 | 生日 | Date | 格式：YYYY-MM-DD |
| 12 | 可用积分 | Number | 整数 |
| 13 | 可用储值 | Number/Decimal | 金额 |
| 14 | 状态 | String | 可用、冻结、注销等 |
| 15 | 注册时间 | DateTime | 时间戳 |
| 16 | 操作 | Action | 查看/编辑/删除等 |

### 操作按钮
| 按钮 | 功能 |
|-----|------|
| 查询 | 执行搜索筛选 |

| 查看 | 查看会员详细信息 |

## 注意事项

1. **权限控制**：不同角色见到的操作按钮和字段可能不同
2. **验证规则**：手机号、邮箱等字段需要格式验证
3. **数据互联**：会员、等级、权益、积分等数据需要保持关联一致性
4. **历史记录**：关键操作（如等级变更、积分变动）需要记录变更日志
5. **批量操作**：支持选中多行进行批量操作（删除、导出等）
6. **导出格式**：支持CSV、Excel等格式导出
7. **时间格式**：系统统一使用YYYY-MM-DD HH:MM:SS格式
